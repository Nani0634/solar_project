## 📝 Customization Guide

### Change Project Details
Edit `index.html`:
- Update **team member names** in the `.team-grid` section
- Update **guide name** throughout the document
- Update **college name** in the cover page
- Update **roll numbers** in the team section

### Change Colors
Edit `style.css` CSS variables:
```css
:root {
  --primary:   #00d4ff;  /* Change to your preferred accent */
  --secondary: #00ff88;  /* Change green theme color */
  --accent2:   #ffd700;  /* Change solar/highlight color */
}
```

### Add More 3D Objects
Edit `script.js`, function `initHeroScene()`:
```javascript
// Add new Three.js geometry here
const newGeo = new THREE.BoxGeometry(1, 1, 1);
const newMat = new THREE.MeshStandardMaterial({ color: 0x00d4ff });
const newMesh = new THREE.Mesh(newGeo, newMat);
scene.add(newMesh);
```
